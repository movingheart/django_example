Django basic example for vote app
===========================
本工程是使用django的一个实例，包含整个简单的投票应用。
在实现这个实例的过程中会涉及到django的各种基本操作---创建project、app、model、template，以及编写相应视图、模板、URL配置等。
使用的版本为django1.8
